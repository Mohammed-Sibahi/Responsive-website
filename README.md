
## Bootstrap Responsive Website

This webpage demonstrates the integration of HTML5, CSS3, Bootstrap (v4.5.3), and Font-awesome. It includes various responsive elements of a webpage: 

 - logo
 - navbar
 - carousel
 - cards
 - jumbotron
 - buttons
 - images
 - icons
 - footer
